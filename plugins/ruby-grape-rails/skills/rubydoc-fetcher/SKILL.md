# Ruby Doc Fetcher

Use `WebFetch` with focused prompts on:

- `guides.rubyonrails.org`
- `ruby-doc.org`
- `rubydoc.info`
- official gem docs or repository docs
- Sidekiq wiki pages

Extract only the relevant API, examples, and caveats for the current task.
